# עדכון TEAMUP

העדכון אינו דורש התקנת חבילות חדשות ואינו משנה את `package.json`.

## בפרויקט שכבר מותקן אצלך

1. עצור את `npm run dev`.
2. העתק מתוך קובץ העדכון את התיקיות `src` ו־`public` אל הפרויקט הקיים ואשר החלפה.
3. ב־Supabase פתח SQL Editor והריץ רק את:

```text
supabase/migrations/002_weekly_polls.sql
```

4. הפעל שוב:

```bash
npm run dev
```

אין למחוק את `node_modules` ואין צורך להריץ שוב `npm install`.

## מה נוסף

TEAMUP, עיצוב רגוע יותר, Arial/Assistant, תיקוני RTL וסטטוסים בעברית, סקר שבועי ראשון–שבת, מסך הקבוצה הכללית, מסך דירוגים, הסברים מלאים לפתיחת משחק, PWA עם הוספה למסך הבית ובדיקת עדכון, ותצוגת טעינה ושגיאות ברורה יותר.
