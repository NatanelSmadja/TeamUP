# TEAMUP – עדכון UX, הרשאות וזרימת משחק

1. עצור את השרת עם Ctrl+C.
2. העתק מהעדכון את התיקיות `src` ו־`supabase` אל הפרויקט הקיים ואשר החלפה.
3. ב־Supabase SQL Editor הרץ את הקובץ `supabase/migrations/003_permissions_workflow.sql` במלואו.
4. הפעל `npm run dev`.

אין צורך למחוק node_modules, אין צורך למחוק package-lock ואין צורך להריץ npm install.

זרימת משחק חדשה:
סקר שבועי → פתיחת הרשמה → סגירת הרשמה → ערבוב קבוצות → פתיחת דירוג לאחר המשחק.

מחיקת משחק והרשאות לחברים זמינות רק למנהל הראשי.
