# TEAMUP V2.8 — תיקון מלא של קבוצות ובקשות הצטרפות

1. העתיקו את `src` ו־`supabase` לפרויקט.
2. הריצו ב־Supabase SQL Editor את `supabase/migrations/012_multi_group_integrity_audit.sql`.
3. הפעילו `npm run dev`.

התיקון כולל:
- חבר פעיל לעולם לא מוצג כ״ממתין לאישור״.
- בקשות הצטרפות נטענות דרך RPC מאובטח עם פרטי השחקן.
- מנהל עם `manage_members` יכול לראות ולאשר בקשות.
- ביטול בקשת הצטרפות.
- מניעת אישור עצמי דרך API ישיר.
- הבטחת Membership פעיל לכל בעל קבוצה.
- קבוצה פעילה נשמרת בנפרד לכל משתמש במכשיר.
