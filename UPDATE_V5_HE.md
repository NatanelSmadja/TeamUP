# TEAMUP V5

## עדכון הפרויקט הקיים
1. עצור את השרת עם Ctrl+C.
2. העתק את תיקיית `src` מתוך קובץ העדכון אל הפרויקט ואשר החלפה.
3. העתק את `supabase/migrations/005_ratings_notifications.sql`.
4. ב-Supabase SQL Editor הרץ את קובץ migration 005 פעם אחת.
5. הפעל `npm run dev`.

אין צורך ב-`npm install`.

## מה נוסף
- מסך דירוג אמיתי עם כרטיס לכל שחקן וכפתורי 1–5.
- אפשרות לבחור MVP ולשנות דירוג קיים.
- התראות לחיצות שמעבירות למשחק, לסקר או לדירוג.
- לחיצה על התראה מסמנת אותה כנקראה ומורידה את המספר האדום.
- בחירת יום בשבוע ביצירת משחק והתאריך הקרוב מחושב אוטומטית.
- חיווי כמה שחקנים כבר דורגו וכמה נותרו.
