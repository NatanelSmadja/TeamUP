# TEAMUP V2.3

## התקנה
1. עצור את השרת עם `Ctrl + C`.
2. העתק את התיקיות `src` ו-`supabase` מהעדכון אל הפרויקט ואשר החלפה.
3. ב-Supabase SQL Editor הרץ את `supabase/migrations/008_team_editing_achievements_emergency.sql`.
4. הפעל `npm run dev`.

אין צורך ב-`npm install`.

## חדש
- נעילת שחקנים בחלוקה, החלפת שני שחקנים, Undo וחלוקה מחדש.
- שיתוף קבוצות דרך תפריט השיתוף או העתקה ל-WhatsApp.
- הישגים אוטומטיים לפי הופעות, MVP ודירוג.
- שחקן החודש לפי דירוגים, MVP והשתתפות.
- כפתור "חסרים שחקנים" ששולח התראה רק למי שעדיין לא נרשם.
