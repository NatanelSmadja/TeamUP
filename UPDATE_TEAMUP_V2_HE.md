# TEAMUP V2 — הוראות עדכון

1. עצור את השרת עם `Ctrl + C`.
2. גבה את הפרויקט או ודא שכל השינויים האחרונים נמצאים ב-GitHub.
3. העתק את התיקיות `src` ו-`supabase` מהעדכון לתוך הפרויקט ואשר החלפת קבצים.
4. ב-Supabase SQL Editor הרץ את:
   `supabase/migrations/006_teamup_v2.sql`
5. הפעל מקומית:
   `npm run dev`
6. בדוק התחברות, דשבורד, משחק, סטטיסטיקות וכרטיס שחקן.
7. פרסם:
   `git add .`
   `git commit -m "Upgrade TEAMUP to V2"`
   `git push`

אין צורך להריץ `npm install` עבור חבילת העדכון.

## מה נוסף
- Dashboard חדש עם המשחק הבא, סקר שבועי, פעילות חיה ומוביל הקבוצה.
- כרטיס שחקן בסגנון ספורט עם Overall, דירוג, MVP והופעות.
- מסך סטטיסטיקות והיכל תהילה.
- מסך פעילות בזמן אמת.
- היסטוריית משחקים.
- קבוצות בצבעים, מדד איזון וגרירת שחקנים במחשב.
- אלגוריתם חלוקה משופר לפי דירוגים ועמדות.
- Realtime גם לפעילות, דירוגים, MVP ותוצאות.
