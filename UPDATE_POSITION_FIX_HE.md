# תיקון תצוגת עמדות

העדכון מתקן שני מצבים:

- ערכים גולמיים באנגלית כמו `goalkeeper` מוצגים עכשיו בעברית.
- אם `preferred_positions` הוא מערך ריק, המערכת נופלת לעמדה הישנה `preferred_position` במקום להציג "ללא עמדה".

## התקנה

1. עצור את השרת.
2. העתק את תיקיית `src` מהעדכון לתוך הפרויקט ואשר החלפה.
3. הפעל `npm run dev`.
4. אין צורך ב-SQL או ב-`npm install`.
