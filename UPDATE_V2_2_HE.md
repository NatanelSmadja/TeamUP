# TEAMUP V2.2

## התקנה
1. עצור את השרת עם Ctrl+C.
2. העתק את התיקיות `src` ו־`supabase` מהעדכון אל הפרויקט ואשר החלפה.
3. ב־Supabase SQL Editor הרץ את `supabase/migrations/007_poll_responses_private_ratings.sql`.
4. הפעל `npm run dev`.

אין צורך ב־npm install.

## מה חדש
- “לא יכול השבוע” נחשב תשובה תקפה לסקר ומסיר בחירות ימים קודמות.
- הצגת מי לא יכול ומי עדיין לא השיב.
- סיכום הסקר ושיתוף מהיר.
- דירוגים והצבעות MVP פרטיים ברמת RLS: כל משתמש יכול לקרוא רק את ההצבעות האישיות שלו.
- טבלאות הדירוג, הדשבורד והפרופילים משתמשים רק בנתונים מצטברים ללא זהות המדרג.
- סטטיסטיקות ציבוריות מתעדכנות אוטומטית וב־Realtime.
