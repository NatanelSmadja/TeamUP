# TEAMUP V4 – הוראות עדכון

1. עצור את השרת עם `Ctrl + C`.
2. גבה את תיקיית הפרויקט הקיימת.
3. העתק מתוך קובץ העדכון את התיקיות `src` ו־`supabase` אל הפרויקט ואשר החלפה.
4. ב־Supabase פתח SQL Editor והריץ את כל הקובץ:
   `supabase/migrations/004_realtime_members_flexible_teams.sql`
5. אין צורך להריץ `npm install` ואין צורך למחוק `node_modules`.
6. הפעל `npm run dev`.

## מה נוסף
- שמות המצביעים בכל יום בסקר ועדכון Realtime.
- שמות הנרשמים למשחק ועדכון Realtime.
- מניעת שגיאת הצבעה כפולה באמצעות upsert.
- חלוקת קבוצות גמישה לפי הכמות שנרשמה בפועל.
- בחירת כמה עמדות בפרופיל.
- משחק פתוח מוצג בעדיפות במסך הבית.
- Tooltips והסברים לפעולות הניהול.
- באנר התקנה ועדכון קבוע בכל האתר.
- הסרה, החזרה ומחיקה קבועה של שחקנים.
- מרכז התראות והתראות דפדפן בזמן שהאפליקציה פעילה.
- עיצוב חדש בגווני כחול־טורקיז ופונט Heebo.

## התראות בטלפון
לאחר העדכון, כל משתמש נכנס לפרופיל ולוחץ "הפעלת התראות". התראות דפדפן מתקבלות בזמן שה־PWA פעילה או מחוברת. Push מלא כשהאפליקציה סגורה לחלוטין דורש שירות Web Push ו־Edge Function נוסף.
